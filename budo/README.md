# budo

Minimal usage for the local libraries in `budo/lib/`:

1. Include headers from `budo/lib/` in your program.
2. Compile your program together with the library `.c` files from `budo/lib/`.
3. Build the bundled demos with `./budo/build.sh`.

API and behavior documentation live in the library headers:

- `budo/lib/budo_graphics.h`
- `budo/lib/budo_audio.h`
- `budo/lib/budo_shader_stack.h`
