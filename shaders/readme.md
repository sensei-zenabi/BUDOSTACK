This folder contains a selection of shaders derived from the libretro
`glsl-shaders` repository (with local parameter tuning).

- crtscreen.glsl: https://github.com/libretro/glsl-shaders/tree/master/crt/shaders/fakelottes-geom.glsl (Unlicense/Public Domain)
- effects.glsl: https://github.com/libretro/glsl-shaders/tree/master/vhs/shaders/vhs.glsl (Unlicense/Public Domain)
- noise.glsl: https://github.com/libretro/glsl-shaders/tree/master/film/shaders/film_noise.glsl (Public Domain)
